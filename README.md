## 📌 О программе

**byeWhiteList** — это Android-приложение на базе NekoBox, которое само находит самый быстрый VPN-сервер. В отличие от обычных VPN-клиентов, оно не просто подключается к случайному серверу, а **тестирует скорость всех доступных серверов** и автоматически переключается на лучший.

### ✨ Возможности

*   **🚀 Умный тест скорости** — пинг и замер реальной скорости (скачивание файла 10MB) для десятков серверов без необходимости подключаться к каждому вручную.
*   **🤖 Автоматический режим** — приложение само в фоне тестирует сервера и, если текущее соединение ухудшается, переключает вас на самый быстрый. Видео при этом не прерывается!
*   **⚡ Максимальная скорость** — тесты заточены под российские CDN (Tele2, Selectel, Hostkey) для быстрого и точного результата даже при заблокированном Cloudflare.
*   **🎯 Умный выбор сервера** — удобный список серверов с сортировкой по скорости и цветовой индикацией (зеленый/желтый/красный).
*   **📊 Мониторинг в реальном времени** — отслеживание скорости текущего соединения прямо на главном экране.
*   **🔄 Автообновление списков** — подписки на актуальные конфигурации обновляются по нажатию кнопки.

## 📥 Установка

1. Скачайте последнюю версию APK из раздела **[Releases](https://github.com/Zhuk001/vpn_whitelist/releases)**.
2. Откройте скачанный файл на вашем Android-устройстве.
3. Разрешите установку из неизвестных источников (система попросит об этом).
4. Готово! Запускайте приложение.

## 🎮 Как пользоваться

1. **Первый запуск:** Приложение автоматически загрузит списки серверов.
2. **Быстрый тест:** Нажмите "🚀 Быстрая проверка Ping", чтобы узнать, какие серверы доступны.
3. **Тест скорости:** Нажмите "⚡ Тест скорости" → "Точный тест (ТОП-50)". Приложение протестирует 50 лучших по пингу серверов и выберет самый быстрый.
4. **Автоматический режим:** Включите "🤖 Автоматический режим". Приложение начнет мониторинг и будет само переключаться на лучший сервер.
5. **Ручной выбор:** Нажмите "🎯 Выбрать сервер вручную", чтобы увидеть все серверы, отсортированные по скорости, и выбрать любой.
