## 📌 Что это?

**byeWhiteList** — это не просто VPN-клиент. Это интеллектуальный помощник, который постоянно следит за качеством вашего соединения и автоматически переключает вас на самый быстрый и стабильный сервер. Забудьте о ручном переключении — приложение сделает всё за вас.

## 🔥 Ключевые возможности

### 🤖 **Авторежим — ваш личный администратор**

| Что умеет | Как работает |
|-----------|--------------|
| **Мониторинг скорости** | Проверяет скорость каждые 5 секунд, собирает данные за 30 секунд |
| **Автоматическое переключение** | Если скорость падает ниже 0.5 MB/s — сам ищет лучший сервер |
| **Плановый ретест** | Раз в 10 минут проверяет, не появился ли сервер быстрее текущего |
| **Умное восстановление** | Если сервер не отвечает — возвращается к предыдущему стабильному |

### 📊 **Двухэтапный тест скорости**

1. **Быстрый пинг-тест** — проверяет все серверы за секунды (15 параллельных запросов)
2. **Точный тест скорости** — тестирует топ-10 серверов по пингу, измеряет реальную скорость

Результат — вы всегда подключены к лучшему серверу, а не просто к тому, у кого меньше пинг.

### 🎯 **Умный рейтинг серверов**

При выборе лучшего сервера мы учитываем:
- **80% — скорость отдачи (upload)** — важно для видеозвонков, игр, мессенджеров
- **20% — скорость загрузки (download)** — важно для просмотра видео, скачивания файлов

### 📱 **Интуитивный интерфейс**

- **Круговые индикаторы скорости** — наглядно показывают текущую загрузку и отдачу
- **Цветовая маркировка** — зелёный, оранжевый, красный — сразу понятно качество
- **Кнопка «Проверить»** — измеряет скорость текущего VPN одним касанием
- **Встроенное руководство** — пошаговая инструкция для новичков

### 🔄 **Автообновление списков серверов**

- При запуске приложения
- Каждый час в фоне (только когда приложение активно)
- Удаление дубликатов автоматически

### 📋 **Диагностика и логи**

- Встроенный просмотр логов
- Отправка отчёта об ошибке одной кнопкой

### 🔔 **Проверка обновлений**

- Автоматическая проверка при запуске
- Уведомление о новой версии с возможностью скачать и установить

## 🎮 Как пользоваться

| Шаг | Что делать | Результат |
|-----|------------|-----------|
| **1** | Установите приложение | Списки серверов загрузятся автоматически |
| **2** | Нажмите **⚡ Тест скорости** → выберите «Точный тест» | Приложение найдёт самый быстрый сервер |

## 🛠️ Технологии

| Компонент | Технология |
|-----------|------------|
| **Язык** | Kotlin |
| **Ядро** | [NekoBox](https://github.com/MatsuriDayo/NekoBoxForAndroid) / sing-box |
| **Сеть** | OkHttp (с прямым клиентом через underlyingNetwork) |
| **База данных** | Room |
| **Параллельность** | Coroutines |
| **Минимальная версия** | Android 5.0 (API 21) |

## 📊 Технические детали

| Параметр | Значение |
|----------|---------|
| **Параллельность пинг-теста** | до 15 потоков |
| **Параллельность теста скорости** | до 3 потоков |
| **Размер тестового файла (download)** | 10 MB |
| **Размер тестового файла (upload)** | 2 MB |
| **Таймаут запроса** | 5 секунд |
| **Порог «медленной» скорости** | 0.5 MB/s |
| **Интервал планового ретеста** | 10 минут |
| **Интервал автообновления списков** | 1 час |
